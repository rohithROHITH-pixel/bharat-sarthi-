# **App Name**: Sarathi News

## Core Features:

- News Display: Display news articles on the home page in chronological order with title, short description, date, and category.
- Admin Authentication: Secure the admin page using Firebase Authentication with email/password, allowing only one admin user.
- News Management: Admin interface to add, edit, and delete news articles stored in Cloud Firestore.
- Real-time Updates: Automatically update the homepage when new articles are posted by the admin.
- Firestore Integration: Store and retrieve news articles from Cloud Firestore, including fields for title, content, category, date, and creator ID.
- Role-based access control: Restricting the modification of content solely to authenticated administrators

## Style Guidelines:

- Primary color: Light sky blue (#87CEEB) to evoke a sense of calm and trustworthiness.
- Background color: Very light grayish-blue (#F0F8FF) to maintain a clean, readable interface.
- Accent color: Soft orange (#FFB347) for category tags and interactive elements.
- Body font: 'PT Sans', a humanist sans-serif for a blend of modern and approachable readability.
- Headline font: 'PT Sans', using a larger size and weight for clear hierarchy.
- Simple, clear icons for categories to aid in quick navigation.
- Clean, grid-based layout optimized for both mobile and desktop, ensuring content is easily accessible and readable on all devices.
- Subtle transitions and loading animations to enhance user experience without being distracting.
